# BioRepro Game Lab 3D

PWA responsivo e imersivo para o projeto de curricularização em Ciências Biológicas.

## Publicar no GitHub Pages
1. Crie um repositório.
2. Envie todos os arquivos desta pasta.
3. Vá em Settings > Pages.
4. Selecione branch `main` e pasta `/root`.
5. Abra o link publicado.

## Conteúdo
- 9 semanas do PWA Game
- Semana 1: escolha de temas
- Semana 2: criação de imagens no Canva/IA
- Botões 3D macios
- Bolhas flutuantes
- BioBot flutuante
- Prompt generator
- Estrutura PWA com manifest e service worker
